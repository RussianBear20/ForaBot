# Setup for Windows/Linux

## Set up conda environment

- Download [Anaconda](https://www.anaconda.com/)
- Open "Anaconda prompt" or command prompt/terminal if installed to path
- cd Forams/forams-bot/setup
- create conda environment
	- Windows run command: conda env create -f environment_windows.yml
	- Linux run command: conda env create -f environment_linux.yml
- Install Serial Port Driver (SiLabs Chipset) from (https://learn.adafruit.com/adafruits-raspberry-pi-lesson-5-using-a-console-cable/software-installation-windows)
